Dasar-Dasar Pengembangan Web

HTML

Bahasa markup yang digunakan untuk membuat struktur dasar sebuah halaman web. HTML digunakan untuk menambahkan elemen-elemen seperti teks, gambar, tabel, formulir, dan lainnya ke dalam halaman web.

CSS

CSS digunakan untuk mendesain dan mengatur tampilan halaman web yang dibuat dengan HTML. CSS mengontrol elemen seperti warna, font, layout, margin, padding, dan posisi.

Javascript

JavaScript adalah bahasa pemrograman yang digunakan untuk menambah interaktivitas pada halaman web. JavaScript memungkinkan kita untuk membuat elemen yang dinamis, seperti animasi, validasi form, manipulasi konten, dan pengambilan data tanpa perlu me-refresh halaman.
