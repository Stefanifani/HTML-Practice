HTML (HyperText Markup Language)

HTML (HyperText Markup Language) pertama kali diciptakan oleh Tim Berners-Lee pada tahun 1991
sebagai bagian dari pengembangan World Wide Web. Berners-Lee merancang HTML untuk
memungkinkan pertukaran informasi di internet melalui dokumen-dokumen yang saling terhubung
dengan tautan. HTML menggunakan tag-tag yang memberi instruksi kepada browser
tentang bagaimana menampilkan konten di halaman web.

Seiring berjalannya waktu, HTML terus berkembang untuk mendukung fitur-fitur baru di web.
Pada tahun 2014, HTML5 dirilis sebagai versi terbaru yang memungkinkan pengintegrasian elemen multimedia seperti
audio, video, dan aplikasi web yang lebih interaktif. Meskipun teknologi web terus berkembang,
HTML tetap menjadi fondasi dasar dalam pembuatan hampir semua situs web yang ada di internet saat ini.
